# GoBusIt — Landing Page

The marketing and informational website for [GoBusIt](https://gobusit-app.vercel.app) — an intercity bus booking platform that lets passengers search trips, pick seats, and book instantly.

---

## 🌐 Live Site

[gobusit.vercel.app](https://gobusit.vercel.app) 

---

## 📄 Pages

| Route | Description |
|---|---|
| `/` | Hero, how it works, feature highlights, CTA |
| `/features` | Deep-dive into passenger and operator features |
| `/about` | The problem GoBusIt solves and the vision behind it |
| `/contact` | Contact form and social links |

---

## 🚀 Tech Stack

- **React 18** + **TypeScript**
- **Vite** — build tooling
- **Tailwind CSS** — styling
- **Framer Motion** — animations
- **React Router** — client-side routing
- **shadcn/ui** — UI component primitives

---

## ⚙️ Running Locally
```bash
npm install
npm run dev
```

App runs on: http://localhost:8080
